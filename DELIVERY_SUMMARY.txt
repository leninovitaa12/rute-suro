ALGORITHM SPECIFICATIONS
================================================================================

A* PATHFINDING
- Manual implementation from scratch
- NetworkX graph library for data structure
- Priority queue for node evaluation
- Admissible heuristic (Haversine distance)
- Time-complexity: O(b^d) where b=branching factor, d=depth

HEURISTIC FUNCTION
h(n) = haversine_distance(n, goal) / max_speed_mps

Admissibility:
- Haversine distance ≤ actual road distance (straight line is shortest)
- Dividing by 70 km/h (maximum speed) ensures cost never exceeded
- Result is always ≤ true cost from n to goal

COST FUNCTIONS
- Fastest mode: Travel time in seconds
- Shortest mode: Distance in meters
- Congestion penalty: 2.5x to 5.0x multiplier on travel time

CLOSURE HANDLING
- Closed edges removed from graph before pathfinding
- A* finds alternative routes automatically
- Returns error if no alternative exists
