================================================================================
                      TROUBLESHOOTING
================================================================================

ERROR: "Table already exists"
SOLUSI: NORMAL! Ini karena "IF NOT EXISTS". Lanjut ke next step.

ERROR: "Column 'status' does not exist"
SOLUSI: QUERY 1 tidak selesai dengan benar!
        - Cek apakah ada error merah saat menjalankan QUERY 1
        - Pastikan tabel 'events' sudah ada di Tables tab
        - Jalankan ulang QUERY 1

ERROR: "Auth user not found" / "Admin user not found"
SOLUSI: Admin user belum dibuat di Supabase Auth!
        - Buka Supabase Dashboard
        - Click "Authentication" > "Users"
        - Click "+ Create new user"
        - Email: admin@rutesuro.com
        - Password: (set yourself)
        - Check "Auto confirm user"
        - Click "Create user"
        - THEN jalankan QUERY 2

ERROR: "Permission denied" saat insert
SOLUSI: Admin user tidak memiliki role 'admin'
        - Pastikan QUERY 2 sudah dijalankan
        - Cek hasil SELECT di QUERY 2 (harus ada admin@rutesuro.com dengan role='admin')
